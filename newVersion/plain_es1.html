    La variable aleatoria XX es normal. Se ha obtenido una muestra de tamaño 154154, con media muestral X‾=9.87\bar{X} = 9.87 y cuasidesviación típica s=4.09s = 4.09. Asumiendo que se trata de una muestra de tamaño grande, calcula el extremo izquierdo de un intervalo de confianza (a,b)(a,\, b) para la media μX\mu_X, con un nivel de confianza del 9090%.
    Utiliza 4 cifras significativas en tu respuesta.


    Solution

    Con α=0.1\alpha = 0.1 el intervalo es X‾±zα/2sn=9.87±1.6454.09154=(9.328,10.41)\bar X\pm z_{\alpha/2}\dfrac{s}{\sqrt{n}} = 9.87\pm 1.645\dfrac{4.09}{\sqrt{154}} = (9.328, 10.41) así que la respuesta es 9.328.
