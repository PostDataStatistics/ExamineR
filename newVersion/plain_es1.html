    La variable aleatoria XX es normal. Se ha obtenido una muestra de tamaño 138138, con media muestral X‾=33.2\bar{X} = 33.2 y cuasidesviación típica s=8.57s = 8.57. Asumiendo que se trata de una muestra de tamaño grande, calcula el extremo izquierdo de un intervalo de confianza (a,b)(a,\, b) para la media μX\mu_X, con un nivel de confianza del 9090%.
    Utiliza 4 cifras significativas en tu respuesta.


    Solution

    Con α=0.1\alpha = 0.1 el intervalo es X‾±zα/2sn=33.2±1.6458.57138=(32,34.4)\bar X\pm z_{\alpha/2}\dfrac{s}{\sqrt{n}} = 33.2\pm 1.645\dfrac{8.57}{\sqrt{138}} = (32, 34.4) así que la respuesta es 32.

  32. Question

    La variable aleatoria XX es normal. Se ha obtenido una muestra de tamaño 99, con media muestral X‾=31.6\bar{X} = 31.6 y cuasidesviación típica s=5.17s = 5.17.     Utiliza 4 cifras significativas en tu respuesta.


    Solution

    Con α=0.05\alpha = 0.05 y k=8k = 8 el intervalo de confianza es (k⋅s2χk;α/22,k⋅s2χk;1−α/22)=(8⋅26.728917.535,8⋅26.72892.180)=(3.492,9.905)\left(\sqrt{\dfrac{k\cdot s^2}{\chi^2_{k;\alpha/2}}}, \sqrt{\dfrac{k\cdot s^2}{\chi^2_{k;1 - \alpha/2}}}\right) = \left(\sqrt{\dfrac{8\cdot 26.7289}{17.535}}, \sqrt{\dfrac{8\cdot 26.7289}{2.180}}\right) = (3.492, 9.905) y la solución es 9.905.
